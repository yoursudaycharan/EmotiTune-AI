# EmotiTune AI 🎵✨

## Overview

EmotiTune AI is an AI-powered emotional music experience that transforms user emotions into immersive music, emotional analytics, AI-generated lyrics, cinematic memories, and interactive soundscapes using Groq AI and the Web Audio API.

The application analyzes user emotions and dynamically generates personalized music experiences in real time.

---

# 🌟 Features

* 🎭 Emotion Detection using AI
* 🎶 Real-Time Music Generation
* 🌊 Live Audio Visualizer
* 🤖 AI Emotional Companion Chat
* 📖 Memory-to-Music Studio
* ✍️ AI Lyrics & Poetry Generator
* 📊 Mood Analytics Dashboard
* ⚡ Smart Music Modes
* 🔊 Web Audio API Sound Engine
* 🌈 Modern Animated User Interface

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/yoursudaycharan/EmotiTune-AI.git
cd EmotiTune-AI
```

---

## 2. Install Dependencies

```bash
npm install
```

---

## 3. Start Server

```bash
npm start
```

---

# 🌐 Run Application

Open the browser and visit:

```bash
http://localhost:3000
```

# 🔑 Groq API Key Setup

Get your free API key from Groq Console:

https://console.groq.com

Paste the API key inside the application.
